# Anastasia / AD Beauty — sorted photo archive

Сортировка сделана визуально по содержимому фотографий и по списку процедур из `treatments.txt`.

## Основные папки

- `01_Lip_Blush` — фото губ / lip blush / semi-permanent lip enhancement.
- `02_Brows_SPMU_Powder_Combo_Stroke` — брови и full-face фото, которые лучше всего подходят под powder/combo/stroke brows и SPMU.
- `03_Eyeliner_Lash_Enhancement_Eyes` — крупные планы глаз, lash line, eyeliner / lash enhancement style.
- `04_Facials_Carbon_Bubble` — carbon facial / bubble facial / skin treatment фото.
- `05_Nails_ARCHIVE_Not_Current_Treatments` — ногти. В загруженном списке процедур ногтей нет, поэтому вынесено в архивную папку.
- `06_Brand_About_Lifestyle` — портреты, lifestyle, About/brand материалы.
- `07_Studio_Process_Materials` — процесс, материалы, студия, trust/behind-the-scenes.

## Специальные папки

- `best6_follow_us` — 6 лучших фото для блока Follow Us / Instagram strip.
- `best6_follow_us_square_ready` — те же 6 фото, но уже подготовленные квадратом 1080x1080.
- `category_covers` — выбранные обложки для категорий с понятными именами файлов.
- `category_covers_square_ready` — квадратные версии обложек.
- `00_reference_contact_sheets` — листы-превью всех фото с индексами, чтобы быстро свериться.
- `00_sorting_index.csv` — таблица: индекс файла → оригинальное имя → папка.

## Важное замечание

Я не увидел явных фото HIFU и IPL-процедур/аппаратов. Поэтому в `category_covers` есть placeholder-файлы:
- `HIFU__placeholder_needs_real_photo.jpg`
- `IPL_Treatments__placeholder_needs_real_photo.jpg`

Их можно временно использовать как визуальные обложки, но перед финальным запуском лучше попросить у Anastasia реальные фото HIFU/IPL.
