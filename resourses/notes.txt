📷 Галерея — добавить фото

Галерея собирается автоматически из папок. Каждая папка = категория (фильтр на странице).

Папка: C:\Users\tupot\Desktop\Projects\Abeauty\public\gallery\

Внутри уже есть: brows, lips, lashes, nails, clinic, aesthetic.

- Добавить фото → просто кинь файл (.jpg, .png, .webp) в нужную папку. Имя любое.
- Новая категория → создай новую папку, например hifu, и положи туда фото — на сайте появится новый фильтр «hifu».
- Удалить фото → удали файл из папки.
- Порядок — по алфавиту имён файлов. Хочешь задать порядок — называй 01-...jpg, 02-...jpg.

▎ Название фильтра = имя папки как есть (маленькими буквами). Хочешь красивую подпись фильтра типа «Semi Permanent» — скажи, добавлю таблицу подписей.

▎ Название фильтра = имя папки как есть (маленькими буквами). Хочешь красивую подпись фильтра типа «Semi Permanent» — скажи, добавлю таблицу подписей.

💅 Тритментсы — добавить / изменить / удалить

Всё в одном файле: C:\Users\tupot\Desktop\Projects\Abeauty\content\treatments.json

Структура одной категории:

{
  "slug": "hifu",                    // адрес страницы: /treatments/hifu (латиница, без пробелов)
  "title": "HIFU",                   // заголовок
  "menuLabel": "HIFU",               // подпись в выпадающем меню
  "image": "/images/skin.jpg",       // фото карточки (файл из public/images/)
  "tagline": "Lift without surgery", // короткая надпись сверху
  "description": "Текст описания...",
  "sections": [
    {
      "title": "Заголовок блока цен",  // можно убрать
      "note": "Примечание курсивом",   // можно убрать
      "items": [
        { "name": "HIFU Facelift", "price": "£499" },
        { "name": "Full Body", "price": "£200", "pack6": "£960" }  // pack6 — цена за 6 сеансов (столбец справа), необязательно
      ]
    }
  ]
}

- Изменить цену/услугу → правишь name / price в нужном items.
- Добавить услугу → добавляешь строку { "name": "...", "price": "£..." } в items.
- Удалить услугу → удаляешь её строку.
- Новая категория целиком → копируешь блок { ... }, меняешь slug, title и т.д. Она сама появится в меню, футере и на странице Treatments.
- Фото карточки → сначала положи картинку в C:\Users\tupot\Desktop\Projects\Abeauty\public\images\, потом впиши её в "image".

⚠️ Это JSON — важны запятые и кавычки. После последнего элемента в списке запятая не ставится. Если где-то ошибёшься — сайт не соберётся (ничего страшного, просто скажешь мне).
